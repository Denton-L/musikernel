Musikernel
==========

Musikernel is a work in progress kernel. The endgoal is for the kernel to be
able to play music using the keyboard as a piano keyboard through the PC
speaker.

Installation
------------

The kernel binary will be called 'musikernel.bin' and can be created by simply
calling 'make'.

To create a bootable ISO image, run 'make musikernel.iso'.

Dependencies
------------

* i686-elf toolchain
* grub (if you wish to create a bootable ISO image)
